# Project Roadmap

## Phase 1: Hardware Stabilization
- Finalize the 16-bit ISA.
- Complete documentation of all opcodes and hardware modules.
- Ensure the CPU is fully simulated and stable.

## Phase 2: Assembler Development
- Build a robust assembler.
- Support labels, directives, and pseudo-instructions.
- Automate binary generation for testing on the CPU.

## Phase 3: Compiler Frontend
- Define the high-level language subset.
- Implement Lexer and Parser.
- Generate AST from source code.

## Phase 4: Compiler Backend
- Implement Semantic Analysis.
- Develop the Code Generator for the custom ISA.
- Run complete end-to-end tests.
