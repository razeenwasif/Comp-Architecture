# 16-Bit RISC CPU & Compiler

This project contains a custom 16-bit RISC CPU architecture
implemented in Digital, along with a custom compiler toolchain
designed to generate assembly and machine code for this
specific architecture.

## Project Structure

- hw/: Contains the hardware design files (.dig) for the CPU.
- compiler/: Contains the source code for the custom compiler.
- documentation/: Contains detailed design docs and roadmaps.
- tools/: Contains assemblers and test binaries.
